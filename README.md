## 🚀 Featured Repositories

### 1. ✈️ [Tourvisto — AI-Powered Travel Agency Platform](https://github.com/Atchaya21007/travel-agency)
An enterprise-grade, full-stack travel booking application that integrates generative AI to streamline trip planning.
* **Key Features:** Google Gemini AI-powered multi-day trip itinerary generator, full admin dashboard with visual business metrics, secure payments via Stripe, and Sentry error telemetry.
* **Stack:** React 19, React Router v7, Tailwind CSS v4, Syncfusion UI Components, Appwrite BaaS, Stripe.

### 🧠 2. [Data-Science-Projects — Machine Learning Portfolio](https://github.com/Atchaya21007/Data-Science-Projects)
A curated collection of 10 production-ready data science and AI applications spanning computer vision, natural language processing, time series forecasting, and anomaly detection.
* **Key Projects Included:**
  * **YOLOv8 Pose Estimation:** Real-time keypoint extraction and action classification (Gym/Dancing/Cooking).
  * **Credit Card Fraud Detection:** Imbalanced class model training using SMOTE and Precision-Recall tuning.
  * **Airbnb Sentiment Dashboard:** Streamlit UI analyzing user reviews comparing BERT, LSTMs, and LLMs.
  * **Resume Parser & Screener:** NLP classification categorizing candidate resumes.
* **Stack:** Python, Scikit-Learn, PyTorch, YOLOv8, TensorFlow, Keras, Hugging Face Transformers, Streamlit, Pandas.
